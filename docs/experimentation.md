# 指标、埋点与 A/B 实验

## 事件边界

采集会话、实验曝光、定位结果、需求提交、推荐开始/结束、结果曝光、证据展开、反馈、多轮补充、降级和重试。

不采集经纬度、完整地址、原始 Query、Dify Answer、证据正文、URL 或 API Key。

## 首个实验

`location_prompt_timing_v1`

- A：页面加载后自动申请定位。
- B：用户打开设置或提交需求后再申请。
- 实验单位：匿名用户 ID。
- 主指标：推荐需求提交率。
- 护栏：有效灵感转化、P95、错误率、空结果率。

自动化 Smoke Test 仅创建两条 demo 会话。其 50% 转化率不能用于判断实验胜负；正式决策需要样本量、SRM、显著性和护栏检查。
